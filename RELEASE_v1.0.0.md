## 🚀 Quick Start

```bash
# Install
curl -sSL https://gitpush.dev/install | bash

# Configure AI
gitpush --ai --configure

# Start using
gitpush  # Interactive mode
gitpush --ai-commit  # AI commits
gitpush --team  # Team features
gitpush --gui  # Launch GUI
```
